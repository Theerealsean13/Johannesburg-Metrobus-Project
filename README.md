<h1>MetroBus Johannesburg - Public Transport Platform</h1>
<h2>Description</h2>
This project is a modern, responsive web platform designed for Johannesburg's MetroBus public transport system. The utility provides commuters with comprehensive route information, live tracking, trip planning, and fare calculations, while also offering an administrative dashboard for managing routes and user feedback.
<br />
<h2>Languages and Utilities Used</h2>
• <b>HTML5, CSS3, JavaScript (ES6+)</b>
• <b>PHP (8.0+)</b>
• <b>MySQL (8.0+)</b>
• <b>Python (Flask - for optional smart features)</b>
<h2>Environments Used </h2>
• <b>XAMPP (Apache, MySQL, PHP)</b>
• <b>Modern Web Browser</b>
